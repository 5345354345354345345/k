> k is the new l, yo

## Repos
![repos-ls](https://raw.githubusercontent.com/supercrabtree/k/gh-pages/repos-k.jpg)

### Usage
hit k, just like you would l (or ls)

```shell
k
```

### Installation
Put `k.sh` somewhere, and source it in your `.zshrc`.

```shell
source ~/path-to/k/k.sh
```
